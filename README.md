## About The Project

MegaShell is a CLI that imitates a Shell (Bash-like) program, it allows you to execute commands in a terminal.

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rodrigov1/MegaShell.git
   ```
2. Make the project and run it
   ```sh
   make run
   ```

### Uninstallation

1. Use make clean
   ```sh
   make clean
   ```
